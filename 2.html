CSS

CSS(스타일 시트, Cascading Style Sheets)는 HTML이나 XML 기반 문서의 표현을 디자인하고 스타일링하기 위한 스타일 시트 언어입니다. CSS는 문서의 레이아웃, 색상, 글꼴, 간격 등의 스타일을 지정하여 웹 페이지를 더 간결하고 시각적으로 매력적으로 만듭니다. 아래는 CSS의 주요 특징과 사용 방법에 대한 간략한 설명입니다:

1. 선택자
CSS는 HTML 요소를 선택하여 스타일을 적용하기 위해 선택자를 사용합니다. 선택자는 특정 HTML 요소를 가리키며, 해당 요소에 스타일을 적용합니다.

2. 속성과 값
선택한 요소에 적용할 스타일 속성과 그 값을 지정합니다. 속성-값 쌍은 세미콜론으로 구분되며, 중괄호로 감싸여 스타일 블록을 형성합니다.
